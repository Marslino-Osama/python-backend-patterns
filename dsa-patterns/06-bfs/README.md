# BFS

## Core Idea

Explore graph levels using a queue, usually for shortest path in unweighted graphs.

## When to Use

Use this pattern when the problem statement shows signs that match this technique.

## Recognition Signs

- Brute force is possible but too slow.
- There is a repeated structure you can optimize.
- The input shape matches this pattern.
